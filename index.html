This is my landing page on GitHub. It is a great time to be interested in tech with drones, self-driving cars, and A.I. on the horizon. I have taught myself quite a bit about computers, at first to squeeze every last bit of processing and bandwidth from my system for advantages in gaming and work. It took quite a while but I am finally in a position to switch careers to a tech job, which I should have done years ago!

I used bootstrap for this page which is a really handy framework for quick design that is responsive. Thumbs up to the twitter bootstrap authors and contributors for their efforts and I can't wait for a Bootstrap 4 beta. This page also uses material-kit by Creative Tim which is an amazing UI for Bootstrap. In the off chance you want to use this theme, feel free to fork the repo, put your content and images or follow the readme if you need.

I wrote about the new iOS beta and although it will still be handy, Apple seems to troll us by announcing that built-in apps can now be removed. And then later adding they programs are not actually removed... only the icons.

If your into php and haven't tried Laravel , test out my simple blog app to get you started. Laravel gives much more life to php with features such as Eloquent Models, Elixir, Facades, Query Builders, Blade Templates and about every other feature you could expect from a cutting edge framework for artisans. After only building one app and spending just a few hours with this framework, I have no intentions of going back.

Laravel_boiler is a quick solution if you just want a working app to build from. This app is a time saver thou because I've set up Gulp/Elixir to copy and compile all CSS/JS/SASS into one file, and PHPUnit is ready for testing with Gulp as well.

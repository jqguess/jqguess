This is my landing page on GitHub. It is a great time to be interested in tech with drones, self-driving cars, and A.I. on the horizon. I've taught myself quite a bit about computers in the last decade, at first to squeeze every last bit of processing and bandwidth from my system for advantages in gaming and work. It took quite a while but I am finally in a position to switch careers to a tech job, which I should have done years ago!
